# DOT Parser — FMCSA Carrier & Insurance Data Tool

Десктопний інструмент для масового збору даних про автотранспортні компанії
(carriers) з відкритих реєстрів FMCSA (US Department of Transportation) за
номером DOT: статус компанії, контакти, адреси, страхові поліси. Плюс
допоміжний веб-скрейпер для пошуку DOT-номерів на сторонньому сайті.

> ⚠️ **Disclaimer.** Модуль `Dot_pars.py` виконує браузерну автоматизацію
> (Selenium + `undetected_chromedriver`) стороннього сайту. Це навчальний /
> внутрішній інструмент, використання на реальних сайтах має відповідати
> їх Terms of Service. Файл `cookies.json`, який генерується цим модулем,
> містить сесійні cookies — **ніколи не комітьте його в репозиторій**.

## Що робить проєкт

- Приймає Excel-файл зі списком DOT-номерів
- Для кожного DOT робить запит до відкритого **Census API** FMCSA
  (`data.transportation.gov`) і забирає: статус, назву компанії, адреси,
  контакти, кількість вантажівок/водіїв, типи вантажів
- Знаходить MC-номер компанії і підтягує до 3 страхових полісів через
  **Insurance API**, сортуючи за датою
- Зберігає все у зведений Excel-звіт з фіксованим набором колонок
- Має окремий режим — парсинг DOT-номерів зі сторінки стороннього сайту
  (`brokersnapshot.com`) через Selenium, з збереженням сесії через cookies
- GUI на PyQt5 з фоновою обробкою (`QThread`), щоб не блокувати інтерфейс
  під час запитів до API

## Технологічний стек

- **PyQt5** — десктопний GUI
- **requests** — робота з Census / Insurance API
- **openpyxl** — читання вхідного та запис вихідного Excel
- **selenium + undetected-chromedriver** — скрейпінг сторінки з DOT-номерами
- **psutil** — гарантоване завершення процесів браузера

## Структура репозиторію

| Файл | Призначення |
|---|---|
| `main.py` | Точка входу, GUI-вікно, вибір Excel-файлу, запуск обробки в окремому потоці |
| `parsed_info.py` | Основна логіка: запити до Census/Insurance API, побудова вихідного Excel |
| `constants.py` | Мапінг полів API → назви колонок, список бажаних колонок, cargo-флаги |
| `Dot_pars.py` | Пошук DOT-номерів на сторінці стороннього сайту через Selenium |
| `cookies.py` | Отримання і збереження cookies для авторизованої сесії браузера |


Потрібні пакети: `PyQt5`, `requests`, `openpyxl`, `selenium`,
`undetected-chromedriver`, `psutil`.

## Запуск

```bash
python main.py
```

1. Натисніть **"📂 Выбрать Excel с DOT"** і виберіть файл з DOT-номерами
   в першій колонці (заголовок в першому рядку)
2. Натисніть **"🚀 Запустить парсинг"** — результат збережеться у `Result.xlsx`
3. За потреби — **"🌐 Парсинг DOT со страницы"**, щоб зібрати DOT-номери
   з переліку компаній на сторонньому сайті
4. **"🍪 Получить Cookies"** — відкриває браузер на 60 секунд для логіну,
   зберігає сесію в `cookies.json` для наступних запусків парсингу
